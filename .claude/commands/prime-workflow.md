# Command: Prime Workflow

## Purpose
Optimize n8n automation flows for efficiency, error handling, and maintainability.

## Read Step
- `.json` workflow exports.
- Custom JS/Python snippets within nodes.
- Webhook trigger definitions.
- Credentials usage logic.

## Analysis
1. **Efficiency:** Eliminate redundant nodes or loops.
2. **Error Recovery:** Verify "On Error" paths and retry logic.
3. **Data Integrity:** Ensure payload schemas are consistent across nodes.
4. **Standardization:** Check against project naming conventions and tagging.

## Report Format
- **Workflow Audit:** High-level view of the flow efficiency.
- **Bottlenecks:** Specific nodes causing delays or failures.
- **Refactoring Steps:** Recommended changes to improve flow.
- **Monitoring Plan:** Indicators to track successful execution.
